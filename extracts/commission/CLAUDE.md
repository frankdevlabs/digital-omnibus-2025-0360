# Commission extracts — working rules

Before adding or editing any file in this directory, read
[`_TRANSCRIPTION_GUIDE.md`](./_TRANSCRIPTION_GUIDE.md) and follow it. It adapts the canonical
[`../council/_TRANSCRIPTION_GUIDE.md`](../council/_TRANSCRIPTION_GUIDE.md) for the **base text** —
the Commission proposal **COM(2025) 837 final** that every Council compromise amends. These extracts
are the diff baseline for the `../council/` files.
